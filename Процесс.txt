1. Создать виртуальное окружение
В папке где хочешь работать через терминал набрать:
python3 -m venv venv

Чтобы активировать виртуальное окружение в терминале нужно набрать

Linux: source venv/bin/activate

Windows: venv\Scripts\activate

Чтобы деактивировать виртуальное окружение в терминале нужно набрать:

deactivate

Работа с пакетами для python через pip:

pip freeze - посмотреть список установленных пакетов

pip install -U lib_name - обновить версию пакета до текущей версии

pip install lib1 lib2 lib3 ... - установить несколько пакетов за раз

pip uninstall lib_name -y - удалить пакет

pip freeze > requirements.txt - создать файл с перечислением зависимостей

pip install -r requirements.txt - установить нужные зависимости в виртуальное окружение указанные в текстовом файле
